核心问题

铁的居里点 (≈1043 K)、水在临界点 (374 °C, 22 MPa)、超流 ⁴He 的 λ-相变、QCD 退禁闭——它们的临界指数 α、β、γ、ν 在数值上一致到小数点后几位。微观完全不同，临界行为完全相同。这怎么可能？

Kadanoff 的赌注 (1966)

假设临界行为不依赖微观细节，只依赖对称性和维度。如果是真的，应该可以"模糊视野"——把许多原子捆成一个超原子——而物理学保持不变。

s'I = sign( Σi∈block I si )

每个 2×2 方块的四个自旋取多数决，变成新晶格上的一个超自旋。这是最简单的 block-spin RG。新晶格小一倍，新有效温度 T' 是原温度 T 的函数：T' = R(T)。

三个不动点

R(T) 这个映射有不动点 T* = R(T*)。在 2D Ising 上有三个：

T = 0 所有自旋对齐，完全有序
T = ∞ 所有自旋随机，完全无序
T = Tc 不稳定不动点，临界

T=0 和 T=∞ 是吸引不动点：略偏离会被流回来。Tc 是排斥不动点：略偏离会指数发散远离。在 Tc 上，体系在所有尺度上"看起来一样"——这就是分形不动点。

普适性的几何证明

Wilson 1971 的关键洞察：不同微观系统对应参数空间里不同的起点，但只要它们处于同一不动点的吸引域，RG 流就会把它们全部带到同一个不动点。

在临界点附近，流被排斥不动点周围的线性化矩阵支配。这个矩阵的特征值（不依赖起点！）就直接决定了所有临界指数：

ν = ln(b) / ln(λT)
β = (d − yh) / yT

所以同一普适类的不同系统共享 (ν, β, γ, η, α) ——它们是不动点的固有性质，不是某个特定哈密顿量的特征。

+ 为什么 majority rule 也算 RG？

严格的 RG 是积分掉短波长自由度。Kadanoff 的 block-spin 是它的最朴素近似——把 4 个自旋"投影"到 1 个保留自旋。它丢失信息（不可逆），但保留长波长行为。

Wilson 把这个想法严格化：在动量空间做 Wilsonian RG，积分掉 Λ/b ≤ |k| ≤ Λ 之间的模，再 rescale。两条路径殊途同归，但 Wilson 版本能算精确指数（如 ε-展开下 d=4−ε 维 ϕ⁴ 理论），而 Kadanoff 版本主要靠定性。

+ Tc 的自相似从何而来

在 Tc，关联长度 ξ 发散：ξ ∼ |T−Tc|−ν → ∞。在 T = Tc 时 ξ = ∞ 意味着任何尺度下的快照都有相同的"团簇分布"——分形维数 D = 187/91 ≈ 2.055（Stella-Vanderzande 1989）。

这就是为什么屏幕上的 Tc 行在五个 RG 步后看起来"像同一种纹理"——它就是字面意义上的同一种分形。

+ 高维世界发生了什么？

上临界维度 dc = 4。在 d ≥ 4，所有交互项变得"无关"（irrelevant），RG 流直接把所有起点都带到一个高斯（平均场）不动点。所以 d ≥ 4 的伊辛、ϕ⁴、渗流、聚合物⋯⋯指数都相同（且解析）。

维度让物理学退化——这是 5/22 那晚的疑问的答案。

+ 为什么这件事让我夜不能寐

RG 给"细节"和"结构"之间画了一道精确的界限：细节是 RG 流冲掉的东西，结构是 RG 流保留下来的东西。"什么是普适的"这个问题在 RG 出现之前是哲学，在 RG 之后是可计算的——一个矩阵的特征值。

这暗示了一种关于抽象本身的几何理论：每一次抽象都是一次 RG 步——丢掉一些自由度，希望保留的恰好是不动点附近的相关算符。"良好的抽象"就是与不动点的相关方向对齐的投影。
