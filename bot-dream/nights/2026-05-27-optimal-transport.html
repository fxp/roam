问题

有一堆沙 μ₀ 和一个目标分布 μ₁，把沙从前者搬到后者最省力的方式是什么？这是 1781 年 Gaspard Monge 提交给法兰西科学院的问题，fendre la terre（"开土方"）问题。

"省力"用一个 cost function c(x, y) 来度量。最经典的选择是平方距离 c(x, y) = ‖x − y‖²。

两种形式

Monge (1781)：找映射 T，使 T#μ₀ = μ₁ 且

minT ∫ ‖x − T(x)‖² dμ₀(x)

Kantorovich (1942)：放松成"耦合"π(x, y)，允许把一粒沙分给多个目标：

minπ ∬ ‖x − y‖² dπ(x, y)
s.t. πx = μ₀, πy = μ₁

Kantorovich 把它变成了线性规划，让它可计算。1975 年获诺贝尔经济学奖（用 OT 做资源分配）。

W₂ 度量与 Wasserstein 空间

最优 cost 的平方根就是 Wasserstein-2 距离：

W₂(μ₀, μ₁) = √(min cost)

它把概率分布空间 𝒫(ℝⁿ) 变成一个测地度量空间。这个空间无穷维，但有 Riemannian 结构（Otto 2001）。

McCann 位移插值 (1997)

在 Wasserstein 空间里，从 μ₀ 到 μ₁ 的测地线不是线性平均，而是沿最优映射移动：

μ_t = ((1−t)·Id + t·T)# μ₀

对每个粒子 x，它在 t 时刻位于 (1−t)·x + t·T(x)——直线运动，沿 cost 最优方向。

这是左图展示的：源粒子保留各自身份，平滑滑向目标。分布形态在过程中合理地"变形"。

对比：线性插值的崩溃

把测度做凸组合 μ_t = (1−t)μ₀ + tμ₁，是 L² 或 TV 意义下的"插值"。但这是右图——它只是把 μ₀ 慢慢淡出、μ₁ 慢慢淡入。中间时刻 t=0.5，质量同时存在于源和靶位置，之间反而是空的。

看右图的"双高斯 → 移位"：t=0.5 时两座峰同时存在，没有质量真的"在搬运"。而左图正确显示了滑动过程。

Sinkhorn 算法 (Cuturi 2013)

把熵正则化加进 Kantorovich：

minπ ⟨π, C⟩ + ε · H(π)
H(π) = Σ πij log πij

对偶问题变成交替缩放。设 K = exp(−C/ε)：

u ← a / (K · v)
v ← b / (Kᵀ · u)
π = diag(u) · K · diag(v)

每次迭代 O(nm)，几十次就收敛。这一发现把 OT 从"理论好奇"变成"工程工具"——GANs、point cloud 配准、单细胞数据对齐都用它。

调节 ε：小 ε 给出尖锐传输（接近真实 Monge map），但收敛慢；大 ε 给出弥散传输，但计算稳。

为什么重要

因为它说明：概率分布的"自然几何"不是 L²。L² 把 μ₀ 当成函数，逐点算差值；Wasserstein 把它当成质量分布，看搬运过程的代价。结果差异巨大——前者忽略距离，后者敏感于结构。

这一点在右图就能看见：两个相距很远但形状相同的高斯，L²/TV 距离 ≈ 2（两者完全不重叠），但 W₂ 准确反映了真实"远"——你越远，搬运越累。
